# ⚡ INSTALLATION

## Debian / Ubuntu / Kali / WSL

```bash
sudo apt update
sudo apt install git curl wget unzip php -y
```

Clone the repository:

```bash
git clone https://github.com/vkumxr/DEADPIXEL.git
```

Move into the directory:

```bash
cd DEADPIXEL
```

Make executable:

```bash
chmod +x deadpixel.sh
```

Run:

```bash
./deadpixel.sh
```
